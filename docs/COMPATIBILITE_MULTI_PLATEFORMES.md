## 📱 COMMENT INSTALLER SUR CHAQUE PLATEFORME

### Android
1. Ouvrir dans **Chrome Android**
2. Menu (⋮) → **"Installer l'application"**
3. Confirmer l'installation
4. Icône apparaît sur l'écran d'accueil

### iOS / iPhone
1. Ouvrir dans **Safari iOS**
2. Bouton **Partager** (□↑)
3. **"Sur l'écran d'accueil"**
4. Personnaliser le nom (optionnel)
5. **"Ajouter"**

### iPad
1. Ouvrir dans **Safari iPad**
2. Bouton **Partager** (□↑)
3. **"Sur l'écran d'accueil"**
4. Icône apparaît sur l'écran d'accueil iPad

### Windows Mobile
1. Ouvrir dans **Edge Mobile**
2. Menu → **"Installer l'application"**
3. Confirmer
4. Apparaît dans le menu Démarrer
