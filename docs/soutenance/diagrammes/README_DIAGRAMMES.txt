Diagrammes soutenance AfriWonder
================================

afriwonder_ecosysteme_complet.md
  - Grande illustration « système complet » : tous les domaines de la super-app
    (social, commerce, finance, messagerie, mobilité, santé, emploi, civic, IA,
    mini-apps, admin…) — le marketplace est une tuile ②, pas le centre du projet.
  - Prévisualiser le Markdown dans l’IDE ou copier les blocs mermaid vers mermaid.live.

use_case_afriwonder_complet.puml
  - Diagramme de cas d'utilisation synthétisé à partir des routes Express
    (backend/src/app.ts et routers sous backend/src/routes/).
  - Copier tout le fichier dans https://www.plantuml.com/plantuml/uml
    ou utiliser l'extension PlantUML (VS Code / Cursor).

Si le rendu est trop large : ouvrir le .puml et commenter des packages
entiers (blocs package ... end) ou des lignes actor --> usecase pour une
version « soutenance » réduite.
